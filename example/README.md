# Example app

## Installation

```sh
yarn
```

For iOS, do the following:

```sh
cd ios
pod install
cd -
```

## Running

```sh
yarn start
```

## Build

You can either build for iOS:

```sh
yarn ios
```

or for Android with 2 variants: `play` or `amazon`.

```sh
yarn android:play
yarn android:amazon
```
